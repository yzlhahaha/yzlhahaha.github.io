Star Wars Universe

In this blog, I will introduce how use “request” to read json format data from Star Wars Universe API, and how to accomplish some of the tasks like get the name of the oldest person in the data, get the list of the titles of all the films each person appeared in with data processing techniques.

  • Step 1: Import necessary Python packages and read in json format data.

  • Step 2: Define the functions needed to process the read-in json format data. For example, the get_nested() function get the all the information in the nested urls, get_titles() function get all title information, and get_birth_num() get all numeric format of the birth_year data.

  • Step 3: Convert the processed data into pandas dataframe.

  • Step 4: Use chain method to get the name of the oldest person.

  • Step 5: Use chain method to get the list of the titles of all the films each person appeared in.
