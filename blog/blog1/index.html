Creating Personal Website with Hugo

Tuesday, Aug 25, 2020| Tags:

This is my first time creating a Hugo website. During the creation of the website, I realized how convenient and powerful Hugo could be for people that are not experts in web developing, but still want to create their own websites for various different kinds of purposes. I hereby share my experience of creating a website, more specifically, a personal portfolio site with Hugo, step by step:

  • The very first step is to install Hugo (For macOS user, just type in ‘brew install hugo’ in terminal to install Hugo with Homebrew). Make sure Homebrew is installed beforehand.

  • Create a new Hugo site with command line ‘hugo new site quickstart’

  • Find an appropriate theme for the newly created hugo site (link: https://themes.gohugo.io/)

  • Add in the theme by first cd into the site directory, then type in command lines: ‘git init’ to first initialize a git repository on local machine, and ‘git submodule add {web URL of the github repository of the web theme}’ to add the theme into the site directory

  • Add the theme into the site configuration by type in command line: ‘echo ‘theme = “{theme name}"‘»config.toml

  • Customize site theme by editing config.toml in site directory accordingly

  • Add new content like posts or projects by type in command line: ‘hugo new {route where new content to be insert followed by file name, e.g: Posts/something.md}’

  • Start the Hugo server by type in command line: ‘hugo server -D’ to start the server in development mode and navigate to the site at appropriate address
